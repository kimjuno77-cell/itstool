# Engineering Tools Portal (통합 엔지니어링 설계 포털 v1.0)

이 포털은 플랜트 설계 및 안전성 검토에 사용되는 **11종의 전문 엔지니어링 계산 툴**을 하나의 모던하고 직관적인 **프리미엄 탭(Tab) 스타일 대시보드**로 통합한 애플리케이션입니다. 

인터넷 연결이 제한된 **사내 보안망, 오프라인 현장, 출장지** 등에서도 완벽하게 단독 작동할 수 있도록 설계되어, 팀원들에게 압축파일 하나만 배포하면 즉시 설치 없이 구동됩니다 (신규 추가된 Streamlit 도구는 인터넷 환경 필요).

## 🛠️ [배포자 전용] 100% 오프라인 패키지 빌드 방법
팀원들에게 배포하기 전, 온라인 CDN(React, Tailwind CSS 등) 의존성을 완전한 로컬 파일로 변환하여 완전 오프라인 패키지를 구축합니다.

1. 인터넷이 연결된 PC에서 본 폴더에 있는 **`setup_offline.bat`** 파일을 더블 클릭하여 실행합니다.
2. 스크립트가 실행되면 외부 라이브러리 파일들을 `libs` 폴더로 다운로드하고, 6개 HTML 내의 경로를 로컬로 자동 수정합니다.
3. 빌드가 정상 완료되었다는 안내가 표시되면 창을 닫습니다.

## 📦 팀원 배포 및 실행 방법

### 1단계: 압축 및 배포
- 구축이 완료된 현재 폴더의 모든 내용(특히 새로 생성된 `libs` 폴더 필수 포함)을 하나의 `.ZIP` 파일로 압축합니다.
- *Tip: `setup_offline.bat` 파일은 배포용이 아니므로 ZIP 압축 목록에서 제외하셔도 무방합니다.*
- 사내 메신저, 공유 드라이브, 이메일 등을 통해 팀원들에게 `.ZIP` 파일을 배포합니다.

### 2단계: 팀원 실행
- 팀원은 배포받은 압축파일을 원하는 로컬 디렉토리에 해제합니다.
- 해제된 폴더 내의 **`index.html`** 파일을 더블 클릭하여 평소 사용하는 웹 브라우저(Chrome, Edge 등)로 엽니다.
- **별도의 서버 설치나 회원가입, 인터넷 연결 없이 즉시 오프라인에서 구동됩니다.**

## 💡 유용한 꿀팁 및 활용 가이드

### 🌓 시력 보호를 위한 Modern Dark Theme
- 대시보드는 모던한 딥 스페이스 다크 테마가 기본 적용되어 장시간 도면 및 계산 검토 작업을 하는 엔지니어분들의 눈 피로도를 최소화합니다.

### 🖨️ 깔끔한 보고서 PDF 저장/인쇄 팁
1. 탭 내부에서 입력값을 모두 기입하고 **"상세 계산서 생성"** 버튼을 눌러 결과 레포트를 띄웁니다.
2. 대시보드 우측 상단의 **[보고서 인쇄 / PDF 저장]** 버튼을 클릭하면 불필요한 대시보드 탭 스킨(네비게이션 메뉴 등)이 깨끗하게 제거된 순수 엔지니어링 계산서 페이지만 인쇄 팝업에 나타납니다.
3. 프린터 대상을 **[PDF로 저장]**으로 지정합니다.
4. **[설정 더보기]**를 클릭하여 **여백을 '없음'** 또는 **'기본값'**으로 맞추고 **배경 그래픽을 '체크'**하시면 기존 HTML 보고서의 프리미엄 스타일이 그대로 깔끔하게 PDF 문서로 영구 소장됩니다.
