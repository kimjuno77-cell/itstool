* Note: Load Calculation Formulas

# : Max Wind Condition (100% Wind)     ## : 50% of Max Wind Velocity

WE: Empty Weight Without Insulation (Metal Weight only)

WT: Total Weight = WE + Liquid Content + Insulation + Attachments

FW: Horizontal Force by Wind = Design Wind Pressure × Projected Area

MW: Overturning Moment by Wind (Calculated at base)

FE: Horizontal Force by Earthquake (Seismic Base Shear)

ME: Overturning Moment by Earthquake (Slab Moment, Ms)

MP: Overturning Moment by Earthquake Shell Pressure (Ringwall Moment, Mrw)

MPB: Overturning Moment by Eq. Bottom Pressure = ME - MP (Calculated approximately)

T: Increased Line Load by MP = Seismic Shell Line Load (V)

Base Shear Check: Friction Resistance (Ffric) = μ × (WT - U). If Ffric < Max(FW, FE), anchors must resist shear.

A: Bottom Plate Static Pressure + Product Static Pressure + Design Internal Pressure + Flolating Roof Static Pressure

B, G: AsBuilt Metal Load by Shell, Fixed Roof & Appurtenances + Live Load - Uplift Load by Internal Pressure

C, H: AsBuilt Metal Load by Shell, Fixed Roof & Appurtenances + Live Load - Uplift Load by Internal Pressure + Load by Wind (# Max Wind)

D, I: (Uplift Load by Internal Pressure - Corroded Metal Load by Shell, Fixed Roof & Appurtenances) / Number of Anchor Bolt

E, J: (Uplift Load by Internal Pressure + Uplift Load by Wind - Corroded Metal Load by Shell, Fixed Roof & Appurtenances) / Number of Anchor Bolt (# Max Wind)

F: Bottom Plate Static Pressure + Design Internal Pressure + Flolating Roof Static Pressure

K: Bottom Plate Static Pressure + Product Static Pressure + Flolating Roof Static Pressure

L, Q: AsBuilt Metal Load by Shell, Fixed Roof & Appurtenances + Live Load

M, R: AsBuilt Metal Load by Shell, Fixed Roof & Appurtenances + Live Load + Load by Wind (# Max Wind)

N, S, AD, AE: Always 0 (Zero)

O, T: (Uplift Load by Wind - Corroded Metal Load by Shell, Fixed Roof & Appurtenances) / Number of Anchor Bolt. (# Max Wind. Enter 0 if negative)

P: Bottom Plate Static Pressure + Flolating Roof Static Pressure

U: Bottom Plate Static Pressure + Test Water Static Pressure + Penumatic Test Pressure + Flolating Roof Static Pressure

V: AsBuilt Metal Load by Shell, Fixed Roof & Appurtenances - Uplift Load by Test Pressure

W: AsBuilt Metal Load by Shell, Fixed Roof & Appurtenances - Uplift Load by Test Pressure + Load by Wind (## at 50% of Max Wind Velocity)

X: (Uplift Load by Test Pressure - AsBuilt Metal Load by Shell, Fixed Roof & Appurtenances) / Number of Anchor Bolt

Y: (Uplift Load by Test Pressure + Uplift Load by Wind at 50% of Max Wind Velocity - AsBuilt Metal Load ...) / Number of Anchor Bolt (## 50% Wind)

Z: Bottom Plate Static Pressure (*1) + Product Static Pressure(*1) + Floating Roof Static Pressure(*1) + Design Internal Pressure + (Overturning Moment by Eq. Bottom Pressure / Section Modulus)

AA: Bottom Plate Static Pressure(*1) + Product Static Pressure(*1) + Floating Roof Static Pressure(*1) + Design Internal Pressure - (Overturning Moment by Eq. Bottom Pressure / Section Modulus)

AB: AsBuilt Metal Load by Shell, Fixed Roof & Appurtenances (*1) + Live Load + Load by Seismic

AC: (Uplift Load by Internal Pressure + Uplift Load by Seismic - Corroded Metal Load by Shell...) / Number of Anchor Bolt

AF: Uplift Load by 1.25 times Test Pressure - Asbuilt Material Load by Shell, Fixed Roof & Appurtenances

AG: Uplift Load by 1.5 times Design Internal Pressure + Uplift Load by Wind - Corroded Material Load...

*1: Vertical Seismic Coefficient shall be taken into account.

1. 자중 및 전단/모멘트 (Weight & Shear/Moment)

항목PDF 내 탐색 위치 및 키워드
WE (kg)'Weight Summary' 테이블의 Empty Weight 값 입력 (단위 kg)
WT (kg)'Weight Summary' 테이블의 Operating Weight 또는 Total Weight 값 입력 (단위 kg)
FW (N) 'Wind Code' 섹션의 Total Wind Force (Fw) 검색. (단위가 ton인 경우 9806.65를 곱하여 N으로 입력)
[도우미] FW (수평 풍하중 전단력) 수동 계산기
항목이 생략된 경우 'WIND LOAD CALCULATION' 표에서 PF(설계풍력), B(외경), H1(높이) 값을 찾아 입력하세요.
PF: N/m² × B: m × H1: m × 저감계수: = 0 N
MW (N-m)'Wind Code' 섹션의 Overturning Moment by Wind (Mw) 검색.
FE (N)'Seismic Code' 섹션 하단의 Final Base Shear (V) 검색. (또는 Cs × Wo의 최종값)
EQ M_rw (N-m)'Seismic Code' 섹션의 Ringwall Overturning Moment (Mrw) 검색.
EQ M_s (N-m)'Seismic Code' 섹션의 Slab Overturning Moment (Ms) 검색.

2. 바닥면 압력 (Area Pressures - Pa 단위)

보통 문서 후반부의 'Foundation Load Summary' 테이블에 명시되어 있습니다.

항목PDF 내 탐색 위치 및 키워드
Btm DeadBottom Dead Load. 수식: tb × γb × 9.81 결과값.
Btm IntInternal Pressure. 수식: Pi 결과값.
Btm LiqStored Liquid. 수식: G × H × γw × 9.81 결과값.
Btm TestHydrostatic Test. 수식: H × γw × 9.81 결과값.

3. 쉘 선하중 (Shell Line Loads - N/m 단위)

'Foundation Load Summary' 테이블의 Shell 구간 하중을 기입합니다.

항목PDF 내 탐색 위치 및 키워드
Shell DeadDead Load 파트의 Shell 라인 하중. 수식: (Ws + Wrs) / (π × D)
Shell LiveMinimum Roof Live Load 파트의 Shell 라인 하중.
Wind MaxWind (horizontal wind component) 파트의 Shell 라인 하중. 수식: 2 × Hs2 × PWS / (π × D)
Shell EQSeismic 파트의 Shell 라인 하중. 수식: [4 × Mrw / D + 0.4 × (Ws+Wrss) × Av] / (π × D)

항목PDF 내 탐색 위치 및 키워드
Tb OpDesign Internal Pressure 파트의 Tb 값 (Uplift가 없으면 0)
Tb Op+WdDesign pressure plus Wind 파트의 Tb 값
Tb TestTest Condition 파트의 Tb 값
Tb EQSeismic 또는 Design pressure plus Seismic 파트의 Tb 최대값
