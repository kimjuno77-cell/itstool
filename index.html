14종 엔지니어링 설계 자동화 툴 및 지식 플랫폼 통합

본 포털 시스템은 구조 설계, 플랜트 기기 기초 하중 검토, HVAC 발열 산정, 동결방지 배관 설계, 폭발 위험 분석, Tank 열손실 및 결로 평가 등 실무 핵심 연산 8가지와 Streamlit 기반의 설계 자동화, FFS 수명 평가, 일정/SOP 도구 4가지, PPT 원본 서식 유지 통합 프로그램, 그리고 새롭게 추가된 Project L&L Knowledge 플랫폼을 하나의 직관적인 탭 환경에 통합하여 설계 신뢰성을 극대화하고 업무 처리를 표준화합니다.

System Update 2026-07-10
배관 및 설비 보온 표면온도 계산서 추가
배관 및 설비 보온 표면온도 산출 및 열손실을 시뮬레이션할 수 있는 계산서 모듈이 신규 통합되었습니다.
System Update 2026-07-02
Project L&L Knowledge 지식 플랫폼 연동
새로운 지식 플랫폼인 Project L&L Knowledge가 좌측 내비게이션 메뉴에 추가되었습니다. 엔지니어링 지식 공유 및 관리에 활용하시기 바랍니다.
System Update 2026-06-23
PPT 원본 서식 유지 통합 프로그램 추가
PPT 병합 자동화 툴(PPT_Merger_Builder_v6) 및 사용 가이드가 신규 등록되었습니다. 좌측 메뉴를 통해 가이드를 확인하고 로컬에서 다운로드 받아 활용하십시오.
System Update 2026-06-16
Tank 열손실/결로 평가 툴 신규 추가
Tank의 열손실, 보온, 히터, 결로/결빙 및 증기압 상승 평가를 위한 설계 툴이 새롭게 통합 탭에 연동되었습니다.
System Update 2026-06-12
Streamlit 공유 설계 웹 도구 4종 통합 연동
인터넷 연결 환경에서 연동하여 설계 효율을 극대화할 수 있는 Streamlit 기반 공유 도구(API 650 Tank Calc, Automation SOP, Schedule EMKO, API 579 Creep Assessment)를 새로 연동하였습니다. 본 탭들은 네트워크망이 활성화된 상태에서만 사용이 가능합니다.
System Notice 2026-06-01
포털 V1.0 공식 탭 패키지 릴리즈
기존에 개별 파일로 분리되어 관리 및 사용이 파편화되었던 7종 설계 계산 툴을 하나로 묶어 정식 배포합니다. 로컬 환경에 맞춰 경량화 및 최적화되어 인터넷 연결이 전혀 없는 **설계 현장, 오프라인 환경, 사내 보안망** 내부에서도 오류 없이 구동됩니다.
Operation Guide 2026-06-01
출력 및 PDF 보고서 저장 시 필수 준수 사항
각 툴 우측 상단이나 메인 헤더의 **"보고서 인쇄 / PDF 저장"** 버튼을 클릭하면, 불필요한 레이아웃이 제외된 깔끔한 인쇄 전용 A4 계산보고서가 로드됩니다. 최종 저장을 위해 인쇄 설정에서 대상을 **'PDF로 저장'**으로 설정하고, 여백은 **'없음'** 또는 **'기본값'**으로 맞추어 보관해 주십시오.
